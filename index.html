Charte du site

Ce site est hébergé par GitHub Page, un fournisseur de serveur frontend gratuit.
Ce projet à but non-lucratif est mis à votre disposition par l'entreprise BusDisplayer (développement open-source).
Les données collectées anonymement sont :
> Les heures de connexion
> Les durées de session
> Le type d'appareil
> Les données globales de vos systèmes (OS, navigateur...)
Celles-ci sont collectées par Google Analyctics dans le but d'effectuer des statistiques
Les données collectées en votre nom sont :
> Votre heure de repas
> Votre date de dernière mise à jour
Sur votre appareil, nous stockons :
> Votre nom d'utilisateur (les identifiants de connexions sont désormais stockés localement sur votre appareil).
> Votre dernière heure de connexion (pour personnaliser les services) [transmise aux serveurs]
Nous nous engageons à ne pas collecter :
> Vos données de géolocalisation
> Les entrées de vos appareils
> Vos mots de passes
> Vos cookies
Ce site interagit avec des API indépendantes succeptibles de comporter des failles de sécurité, à savoir :
* OPEN MÉTÉO pour connaître les informations météo
* APOD pour obtenir une image de l'espace chaque jour (Nasa)
* Google Sheet pour le backend et datastore (100 % sécurisé)
* L'API de traduction open-source Mymemory
* W3 School pour le style général de la page
En cas de demandes, adressez-vous au service de développement, n'hésitez pas à faire part de vos suggestions par ailleurs !
Attention : Ce site respecte votre vie privée mais pas tous ses homologues ! Sur Internet, restez vigilant !
